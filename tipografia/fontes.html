TIPOGRAFIA

Tipos de fontes

Podemos colocar tipos de fontes em nosso documento, para colocar uma fonte usamos Font-Family. Também temos que lembrar que existem dois modelos de fonte, são "serif" e "sans-serif". A "serif" são tipos de fontes que tem um guia de leitura (que não é muito boa para ler em sites), e a "sans-serif" são tipos de fontes que não tem guia para leitura (que são boas para ler em sites).

Tamanho de fontes

Para medidas de fonte ("Font-size") usamos só "px" e "em" (Recomendação da W3c)

Peso, estilo e shorthand font

O peso da fonte se usa com o parametro Font-weigth, com ela podemos colocarmos nos seguintes valores: lighter, normal, bold e bolder.

Para o estilo de fonte se usa o parametro Font-style, com ela podemos colocarmos nos seguintes valores: italic, normal, oblique, inherit, initial e unset

No shorthand se usa o paramentro Font, ela é a simplificação para usar todos os parametros de fontes. Para usar é muito importante saber o padrão da sequencia, que é na seguinte ordem: font-style, font-weight, font-size, font-family.

TEXTO

Alinhamento

Para alinhamento usamos o parametro text-align com os determinados valores: "center", "justify", "right" e "left". o "center" é para alinhamento ao meio, "justify" é para alinhar o texto para os dois lados(mais usado nas tags "p"), "right" é alinhamento para direita e o "left" é para alinhar a esquerda.
